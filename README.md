## Homework 1 — Расстояние Левенштейна

Консольное приложение на C# для вычисления расстояния Левенштейна между двумя строками.

### Как работает

1. Программа запрашивает первую строку.
2. Если введено `exit`, программа завершается.
3. Иначе запрашивается вторая строка.
4. Выводится расстояние Левенштейна.
5. Процесс повторяется в бесконечном цикле.

### Запуск

```bash
cd homework1
dotnet run
```

## Homework 2 — Учёт музеев и экспонатов

Консольное приложение на C# для управления базой данных музеев и экспонатов.

### Стек

- **БД**: SQLite (`Microsoft.Data.Sqlite`)
- **Паттерн**: Fluent Interface для построения отчётов
- **.NET**: 8.0

### Структура таблиц

| Таблица | Поля | Роль |
|---|---|---|
| `museum` | `museum_id`, `museum_name` | Справочник (один) |
| `exhibit` | `exhibit_id`, `museum_id`, `exhibit_name`, `exhibit_year` | Основная (много) |

### Функциональность

- Добавление, редактирование, удаление музеев
- Добавление, редактирование, удаление экспонатов
- Отчёты через Fluent Interface: фильтрация по музею/годам, сортировка, статистика

### Запуск

```bash
cd homework2
dotnet run
```

## Homework 3 — Веб-приложение для учёта музеев и экспонатов (ASP.NET Core MVC)

Веб-приложение на базе ASP.NET Core MVC с использованием Entity Framework Core и SQLite для управления базой данных музеев и их экспонатов.

### Стек

- **Фреймворк**: ASP.NET Core MVC (C#)
- **БД / ORM**: SQLite + Entity Framework Core 8
- **Вёрстка**: HTML/CSS + Bootstrap 5
- **.NET**: 8.0

### Функциональность

- Полный набор CRUD-операций (Создание, Чтение, Обновление, Удаление) для:
  - Музеев (справочник).
  - Экспонатов (основная таблица с привязкой к музеям).
- Аналитические отчёты (через LINQ к EF Core):
  - Полный список экспонатов с их музеями (Include).
  - Подсчёт количества экспонатов в каждом музее (GroupBy + Count).
  - Вычисление средней оценочной стоимости экспонатов по музеям (GroupBy + Average).
- Автоматическое создание и заполнение базы данных начальными тестовыми данными (Seeding).

### Запуск

```bash
cd homework3
dotnet run
```
